# My Simple Website

A responsive website showcasing a modern image gallery built with HTML and CSS.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth hover effects
- **Image Gallery**: Grid-based gallery with 20 images
- **Navigation**: Easy navigation between Home, About, and Contact pages
- **Accessibility**: Semantic HTML and basic accessibility features

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, Media Queries)

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd my-simple-website
   ```

2. Open `index.html` in your web browser to view the website.

## Folder Structure
